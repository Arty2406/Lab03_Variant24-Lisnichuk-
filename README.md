Лабораторная работа №3 по предмету "Алгоритмизация и программирование"


Цель работы
Изучить классические алгоритмы, освоить методы анализа временной сложности, приобрести навыки разработки приложений с графическим интерфейсом и модульного тестирования.
Постановка задачи
Разработать приложение на C# с использованием Windows Forms, содержащее:
1.	Реализацию двух алгоритмов согласно варианту задания с самостоятельным написанием кода без использования встроенных функций.
2.	Графический интерфейс для ввода данных, запуска алгоритмов и отображения результатов.
3.	Измерение времени выполнения каждого алгоритма с использованием класса Stopwatch.
4.	Подсчет количества выполненных элементарных операций для каждого алгоритма.
5.	Визуализацию результатов сравнения в виде таблицы или графика.
