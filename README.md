# Лабораторная работа №3 по предмету "Алгоритмизация и программирование"


## Что выполняет данная программа?


Программма генерирует различное количество случайных промежутков. Также при нажатии на соответствующие кнопки программа выполняет действия: слияние промежутков методом перебора и оптимальным методом с использованием сортировки с замером времени выполнения. После вы можете увидеть, насколько время дного времени отличается от другого.


## Архитектура проекта


Пространство имён Lab03_Variant24_Lisnichuk_ - группирует связанные компоненты проекта и предотвращает конфликты имён с другими проектами или библиотеками.


Форма ProjectForm - главное меню всей лабораторной работы. Содержит в себе четыре кнопки, при нажатии на которые совершается выполнение привязанных к ним (кнопкам) задач.


Класс IntervalMerger – содержит в себе основные функции вычисления для двух алгоритмов. Можно сказать, что это основная функция программы.


Класс Interval– содержит в себе функциb для «конструирования» интервалов в виде [10 - 20].


## Технические требования


Данный проект был разработан на языке C# в интегрированной среде разработки Visual Studio. Microsoft Visual Studio поддерживает несколько фреймворков от Microsoft. Среди них — .NET Framework, .NET Core, UWP (Universal Windows Platform) и Silverlight.


Чтобы скачать и установить Visual Studio для Windows, нужно: 


1. Перейти на официальный сайт Visual Studio.


2. Выбрать подходящую редакцию: Community (бесплатная), Professional или Enterprise.


3. Скачать и запустить установщик Visual Studio Installer.


4. В окне установщика выбрать нужные рабочие нагрузки: например, «Разработка классических приложений .NET» — для десктопных приложений Windows.


5. Выбрать дополнительные компоненты в правой панели.


6. Указать место для установки (рекомендуется SSD-диск для повышения производительности).


7. Нажать «Установить» и дождаться завершения процесса.
 

Совет: после установки рекомендуется выполнить первоначальную настройку: войти в учётную запись Microsoft при первом запуске, выбрать предпочтительную тему оформления (светлая/тёмная), настроить сочетания клавиш в разделе «Инструменты» → «Параметры» → «Клавиатура», установить полезные расширения через меню «Расширения» → «Управление расширениями». 


Важно: Visual Studio требует совместимой версии Windows (Windows 10 или позже). Подробные спецификации, включая требования к памяти, месту на диске и процессору, доступны на официальном сайте Visual Studio. 


## Как сделать клонирование репозитория?


Для того, чтобы клонировать репозиторий именно через Visual Studio нужно: открыть приложение Visual Studio → в верхнем меню выбрать пункт Git → из выпадающего меню выбрать пункт "Клонировать репозиторий" → в окно "Расположение репозиторя" вставить ссылку на репозиторий, который вы хотите клонировать (например, ссылка данного репозитория: https://github.com/Arty2406/Lab03_Variant24-Lisnichuk-/tree/master) → указываете нужный путь → нажимаете в правом нижнем углу "Клонировать".
